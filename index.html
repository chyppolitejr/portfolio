This project is simple weather dashboard intended to provide the end-user with a simple way to retrieve weather forecasts using the openweather.org api. The application will store the last 10 cities that were search to make it easy for a person to search the same city again without having to type in the information. The dashboard provides the current weather and the weather for the next five days.
